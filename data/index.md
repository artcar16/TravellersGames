# The Travellers — Tournois de Backgammon

Base de données générée automatiquement par l'application. Ne pas éditer à la main.

Chaque tournoi est stocké dans un fichier `data/tournament-<id>.md` contenant
un récapitulatif lisible (joueurs, tableaux, règlement) ainsi qu'un bloc
`json` machine pour le rechargement par l'application.
